## Task 1.3 – React Counter Component

A simple React functional component demonstrating state management using hooks.

### Features:
- Displays a counter value
- Increment button to increase the count
- Reset button to reset the count to zero
- Implemented using functional components and hooks

### Tech Used:
- React
- JavaScript (ES6)
- React Hooks (`useState`)

### How it works:
- `useState` is used to manage the counter state
- Button click events update the state
- UI re-renders automatically on state change
